🌟 软件介绍

Cherry Studio 是一款支持多个大语言模型（LLM）服务商的桌面客户端，兼容 Windows、Mac 和 Linux 系统。

✨ 主要特性

🌐 多样化 LLM 服务支持

  • ☁️ 支持主流 LLM 云服务：OpenAI、Gemini、Anthropic、硅基流动等
  • 🔗 集成流行 AI Web 服务：Claude、Peplexity、Poe、腾讯元宝、知乎直答等
  • 💻 支持 Ollama、LM Studio 本地模型部署

🤖 智能助手与对话

  • 📚 内置 300+ 预配置 AI 助手
  • 🤖 支持自定义创建专属助手
  • 💬 多模型同时对话，获得多样化观点

📄 文档与数据处理

  • 📄 支持文本、图片、Office、PDF 等多种格式
  • ☁️ WebDAV 文件管理与数据备份
  • 📊 Mermaid 图表可视化
  • 💻 代码高亮显示

🛠️ 实用工具集成

  • 🔍 全局搜索功能
  • 📝 话题管理系统
  • 🔤 AI 驱动的翻译功能
  • 🎯 拖拽排序
  • 🔌 小程序支持
  • ⚙️ MCP(模型上下文协议) 服务

🎨 优质使用体验

  • 🖥️ Windows、Mac、Linux 跨平台支持
  • 📦 开箱即用，无需配置环境
  • 🎨 支持明暗主题与透明窗口
  • 📝 完整的 Markdown 渲染
  • 🤲 便捷的内容分享功能

⚙️ 初始配置

1. 首次启动

打开 Cherry Studio 后，您会看到类似的初始界面：

2. 🔑 API 配置（重要）

⭐ 注意：API 配置是使用 Cherry Studio 的关键步骤，正确的设置将确保您能够顺利使用各种 AI 模型。

2.1 进入 API 设置页面

  1. 点击左侧菜单的 设置 图标
  2. 选择 添加 选项，添加提供商
  3. 在右侧窗口将显示 API 配置界面

2.2 填写 API 信息

在配置页面中，您需要填写以下关键信息：

配置项 示例值 说明
🔗 API 地址 https://api.cube004.com API 服务器地址，必须包含完整的 URL
🔑 API 密钥 当然是找站长要啊，难道这里有？ 访问 API 的授权密钥，请妥善保管
🤖 默认模型 claude-opus-4-20250514 设置为默认使用的 AI 模型

2.3 模型配置

模型添加步骤：

  1. 初次配置时，模型列表为空
  2. 点击 添加模型 按钮
  3. 在弹出的对话框中输入 claude-opus-4-20250514
  4. 点击确认完成添加

💡 提示： claude-opus-4-20250514 (简称 Claude-4o) 是目前性能最佳的 AI 模型，建议优先使用。其他模型可在 https://api.cube004.com 的”模型定价”部分查看，但请注意部分模型可能处于维护状态。

3. 验证 API 连接

完成 API 信息填写后，必须验证连接是否正常：

  1. 点击 API 密钥右边中的 检测 按钮
  2. 选择刚刚添加的模型
  3. 系统将自动尝试连接 API 服务器
  4. 等待几秒钟，查看测试结果

测试结果说明：

  • ✅ 成功：系统显示 “连接成功！” 的绿色提示

    • 此时 API 配置完成，可以开始使用 AI 功能
    • 建议创建一个简单测试对话，确认响应正常

  • ❌ 失败：系统显示错误信息

    • 检查 API 地址是否输入正确（包括 https:// 前缀）
    • 验证 API 密钥是否有效（复制粘贴时注意不要有多余空格）
    • 确认网络连接正常，可访问外部服务
    • 如问题持续，联系管理员获取支持

❓ 常见问题

Q1: API 连接失败怎么办？

🔧 解决方案：

  • 检查 API 密钥是否正确
  • 确认网络连接正常
  • 验证 API 地址无误
  • 找站主
  • 可能在维护

Q2: 对话内容会被保存吗？

🔒 隐私说明：

  • 本地对话历史加密存储
  • API 请求使用 HTTPS 加密
  • 可随时清除所有数据
